# GreenBin

GreenBin is a waste classification web application.

The user uploads an image of a waste item, the system classifies the item using a machine learning model, and then recommends the correct recycling bin or nearby recycling point.

## Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- ML: Python image classification model
- Data: Local JSON file for recycling points
